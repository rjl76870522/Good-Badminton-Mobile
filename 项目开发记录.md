## 一、项目概况

基于计算机视觉的羽毛球比赛视频分析系统。使用 YOLO 姿态检测 + 羽毛球检测，输出标注视频、热力图、散点图和统计数据。

**代码地址:** `C:\Users\lanld\Good-Badminton`

#### 已完成的接口

| 接口 | 方法 | 说明 |
|---|---|---|
| `/` | GET | 完整功能测试页面 |
| `/api/upload/video` | POST | 上传视频文件 |
| `/api/upload/template` | POST | 上传模板图片 |
| `/api/templates/detect` | POST | 上传模板+自动检测角点 |
| `/api/templates/extract/{video_id}` | POST | 从视频截取最佳帧做模板 |
| `/api/templates/file/{template_id}` | GET | 获取模板图片文件 |
| `/api/templates/{template_id}` | GET | 获取模板信息及角点 |
| `/api/templates/{template_id}/corners` | PUT | 手动修正角点 |
| `/api/analyze` | POST | 提交分析任务 |
| `/api/status/{task_id}` | GET | 查询分析进度 |
| `/api/result/{task_id}` | GET | 获取分析结果 |
| `/api/results/{task_id}/{filename}` | GET | 下载结果文件 |
| `/api/jobs` | GET | 历史任务列表 |
| `/api/jobs/{task_id}` | DELETE | 删除任务 |

#### 网页测试页功能
- 视频选择 + 模板选择或从视频截取
- 自动检测角点并显示预览图
- 手动标记角点（在图片上点4个点）
- 分析设置（模型/语言/显示开关）
- 开始分析 + 进度条
- 结果展示（视频/热力图/散点图/统计）
- 历史任务列表（可点击查看）
- localStorage 状态持久化（刷新不丢失）

## 三、关键技术细节

### 核心依赖
- Python 3.10+, PyTorch 2.5, OpenCV, Ultralytics YOLO, ONNX Runtime
- FastAPI + Uvicorn（后端）, CustomTkinter（桌面版）

### 分析引擎入口
```python
from analyze_badminton import analyze_badminton_video
result = analyze_badminton_video(
    video_path="...",
    template_path="...",
    show_display=False,
    progress_callback=progress_cb,
    court_corners=corners,  # 跳过GUI标注
)
```

### 性能（CPU 模式）
- demo.mp4（14秒）：~165秒处理，~11x慢放
- 石宇奇片段（35秒）：~709秒处理，~20x慢放
- 每帧：姿态检测 ~0.2s，羽球检测 ~0.3s

## 四、已知问题

| 问题 | 说明 |
|---|---|
| FFmpeg 转码 | 输出视频 H.264 转码有时失败，fallback 到原始 mp4v |
| Gradio WebUI | Gradio 6.x 启动检查 502，需 `ssr_mode=False` + try-catch |
| 防火墙 | 关防火墙才能局域网访问，建议单端口放行 |
| GPU | 当前用 CPU 推理，速度慢。有 GPU 可提速 10-20x |
